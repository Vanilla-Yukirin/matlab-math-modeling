# PART1 Matlab介绍


## 为什么选MATLAB？

左右分两栏，一边是matlab一边是python

matlab：

- 功能丰富：**内置**数学计算函数和工具箱（各领域的函数库）、交互式工作区、工程领域标准
- 多工具箱：不需要会环境配置，安装好后开箱即用
- 文档齐全，内置帮助文档（按F12即可查看函数详细用法），相比去问AI会更加精确且全面
- 付费软件，昂贵的授权费，每个工具箱还需要单独付费[脚注：其实我们学校之前是有订阅过的，但是嘛，这里有个悲伤的故事……]，很多学校没有资金订阅

python：

- 功能丰富：有numpy、sklearn、pandas、matplotlib等科学计算库、jupyter notebook交互式工作区，但是需要会环境配置和手动安装
- 多开源包：需要会环境管理/虚拟环境（anaconda、uv、venv等）
- 教程齐全：作为一门热门语言，网络上教程很多，问LLM也能得到不错的答案
- 免费软件：免费、开源、社区活跃，每个人都能自由的使用（freeware）



**为什么数模初学者用MATLAB？**
MATLAB 是工程领域的计算器+画图板+方程求解器——5分钟从数据到图表，无需折腾环境配置。安装完立即算矩阵、画图、解方程，让新生专注算法而非工具；而 Python 需要先学虚拟环境管理，对完全零基础的大一同学是额外负担。但自己装 Python 的编程环境常见的环境问题（依赖冲突、版本不兼容）往往劝退初学者。




## Matlab版本选择

Matlab相邻版本之间差别并不大，且近年更新的高级功能做数模一般用不上，所以只要不用过于老的版本（推荐使用$\ge2016b$），语法都是通用的。

ab尾缀的含义：
- a：上半年发布的版本，通常在3月份发布
- b：下半年发布的版本，通常在9月份发布
- 通常来说，一个a版本发布后，经过几个月的使用和反馈，MathWorks会修复一些Bug，并在同年秋季的b版本中使其更加稳定。
所以推荐使用b尾缀的版本

接下来的教程是基于Matlab R2024b的，如果你的版本不是2024b也没有关系，操作完全相同。

### 命令行窗口
命令行窗口的左上角会有三个大于号：`>>>`，表示等待用户输入命令。
命令行窗口是Matlab最常用的窗口，用于输入命令，执行命令，查看结果。
可以直接在命令行窗口中写一些简单的代码，也可以把命令行窗口当作一个全能计算器。

### 文件命名规范
在左侧工作区内右键，点击新建脚本
MATLAB 文件名必须以字母字符开头，最多可以包含 63 个字母数字字符或下划线。
也就是说
- 不能以数字开头
- 不能包含中文
- 不能用减号`-`，只能用下划线`_`

养成良好的文件命名习惯，在数模比赛中快速的找到文件
在数模中，由于会有很多道子问题，所以不妨简单的命名为
- `problem_1_GA.m` 问题一，使用遗传算法
- `problem_2_draw.m` 问题二，画图相关代码

### 运行代码

新建`HelloWorld.m`文件，写入
```matlab
n=input("请输入一个数字：");
n
fprintf("Hello, World!\n")
```
然后可以用下列方式运行代码：
1. 上方的“运行”按钮
2. 全选代码，右键点“在命令行窗口中运执行所选内容F9”
3. 鼠标放在代码行号的左侧，出现蓝色条，左键点击
三种方法效果一样，作用不同。个人推荐1和3

点击运行后，会在命令行窗口中看见“请输入一个数字：”，输入数字并回车，程序会输出方才输入的数字，并输出`Hello, World!`

#### **1.4 矩阵魔法秀：大一新生的线性代数预习 —— 15分钟**  
- **拒绝枯燥**：用视觉化案例替代理论  
  - **矩阵乘法**：  
    ```matlab  
    A = [1 2; 3 4];  % 2x2矩阵  
    v = [1; 0];      % 向量  
    plot([0,v(1)],[0,v(2)],'b-','LineWidth',2); hold on;  
    plot([0,(A*v)(1)],[0,(A*v)(2)],'r--','LineWidth',2); % 展示线性变换  
    ```
    → **结论**：*“矩阵乘法 = 坐标变换”*（配几何动画）  
  - **斐波那契加速**（矩阵快速幂思想）：  
    ```matlab  
    F = [1 1; 1 0];  
    n = 10;  
    result = F^n; % 直接计算第n项（对比循环10000次）  
    ```
    → **关键话术**：*“MATLAB让算法提速1000倍，你只需要写一行”*

### **第二部分：MATLAB玩转数学模型（60分钟）**  
#### **2.1 线性回归：预测奶茶销量 —— 35分钟**  
- **为什么选它**：  
  - 无复杂公式（用工具箱隐藏数学）  
  - 生活化案例（新生共鸣）  
  - 产出直观图表（成就感强）  
- **全流程实操**：  
  ```matlab  
  % 1. 准备数据（课前生成csv发群）  
  data = readtable('milktea_sales.csv'); % 两列：气温, 销量  
  % 2. 一键拟合（强调：不用懂原理！）  
  mdl = fitlm(data.气温, data.销量);  
  % 3. 可视化结果  
  plot(mdl); % 自动生成拟合曲线+置信区间  
  % 4. 预测明天销量（25℃）  
  predict(mdl, 25)  
  ```
- **关键教学点**：  
  - 用`mdl`变量查看R²（解释“0.85=85%可信”）  
  - 拖动图表点实时更新模型（交互演示）
